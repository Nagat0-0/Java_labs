üçï Food Delivery API

RESTful API for managing restaurants, menu items, customers, and orders.

üçî Menu Items

GET /api/menuitems - Get all menu items
GET /api/menuitems?maxPrice=X - Filter by max price
GET /api/menuitems/{name} - Get item by name
POST /api/menuitems - Create new item
PUT /api/menuitems/{name} - Update item
DELETE /api/menuitems/{name} - Delete item

üè† Restaurants

GET /api/restaurants - Get all restaurants
GET /api/restaurants?cuisine=X - Filter by cuisine type
GET /api/restaurants/{name} - Get restaurant by name
POST /api/restaurants - Create new restaurant
PUT /api/restaurants/{name} - Update restaurant
DELETE /api/restaurants/{name} - Delete restaurant

üë§ Customers

GET /api/customers - Get all customers
GET /api/customers?address=X - Filter by address
GET /api/customers/{id} - Get customer by ID (First Last)
POST /api/customers - Create new customer
PUT /api/customers/{id} - Update customer
DELETE /api/customers/{id} - Delete customer

üì¶ Orders

GET /api/orders - Get all orders
GET /api/orders?status=X - Filter by status
POST /api/orders - Create new order

Built with Java Servlets, Jackson, SLF4J/Logback
